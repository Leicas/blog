# Alibre


Advantages



	  * Cheap compared to other CAD software (*cough* Solidworks *cough*)

Disadvantages

	  * UI doesn't look as nice
	  * Some keyboard/mouse commands are non-intiative
	  * Some sketch object constraints are hard/impossible to remove once added
