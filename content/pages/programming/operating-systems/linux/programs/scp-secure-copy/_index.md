# Overview




scp (secure copy) is a UNIX command-line program.




# Copy File From Local To Remote



    
    $ scp ~/myfile.txt username@187.23.43.3:~/




# Copy Directory From Local To Remote



    
    $ scp -r ~/myfolder username@ip_address:~/folder_on_remote



    
    $ scp -r ~/myfolder username@ip_address:~/folder_on_remote
