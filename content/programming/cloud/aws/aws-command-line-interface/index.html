S3

Listing Files

To list buckets:

{{< highlight bash >}} $ aws s3 ls {{< /highlight >}}

Copying Files

{{< highlight bash >}} $ aws s3 cp {{< /highlight >}} {{< highlight bash >}} # Copy local file to S3 $ aws s3 cp my_file.txt s3://<my_bucket>/<my_prefix> # Copy file from S3 to local $ aws s3 cp s3://<my_bucket>/<my_prefix> my_file.txt {{< /highlight >}}

Syncing Files

Syncing can copy a group of files which all share a similar prefix (or part of a prefix), so you can treat the prefixes almost like directories in this case.

{{< highlight bash >}} $ aws s3 sync <source> <destination> {{< /highlight >}} {{< highlight bash >}} # Sync all objects with the prefix "my_prefix/" to the current directory $ aws s3 sync s3://my_bucket/my_prefix/ ./ {{< /highlight >}}
